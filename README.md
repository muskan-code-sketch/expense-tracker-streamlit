💸 Personalised Expense Tracker

The Personalised Expense Tracker is an interactive and user-friendly web application developed using Python and Streamlit, designed to help users efficiently manage, organize, and analyze their daily expenses. 📊

This application provides a seamless experience for tracking financial activities by allowing users to add, save, and load expense data effortlessly. It helps users maintain a structured record of their spending habits and supports better financial decision-making. 💡

To enhance usability and insights, the application includes multiple data visualization features such as charts and graphs 📈, enabling users to clearly understand their spending patterns, identify trends, and monitor their financial behavior over time.

🛠️ Tech Stack
  🐍 Python
  ⚡ Streamlit
  📊 Data Visualization (Matplotlib / Pandas)
